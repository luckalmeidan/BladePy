Detailed Description

Filee that contains the class ShapeManager that is inherited by Core.BladePyCore for Shape Control purposes.

The class has a object compost in Core.BladePyCore. In this file the shape color definitions are also defined and imported through the GUI for painting shapes and for painting the TreeView widget icons.
